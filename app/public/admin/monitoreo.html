¿Qué se monitorea?

Este módulo lee métricas en tiempo real desde las vistas dinámicas de Oracle (V$SESSION, V$LOCK, V$TRANSACTION, V$SQL y V$SYSMETRIC). Sirve para detectar carga alta, sesiones bloqueadas, locks activos, transacciones abiertas y consultas costosas.
